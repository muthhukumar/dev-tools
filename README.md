
# List of utility tools

A list of utility tools I created.


## Installation

```bash
  git clone https://github.com/muthhukumar/dev-tools.git
  cd dev-tools
  npm install
```
    
## Running Tests

To run tests, run the following command

```bash
  npm run test
```

To run test in watch mode, run the following command

```bash
  npm run test-watch
```
